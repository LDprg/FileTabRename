# FileTabRename
A Sublime Text 3 Package for renaming files via Tab Context!

Save the file to your Packages\User folder.

Windows: C:\Users\"yourUserName"\AppData\Roaming\Sublime Text 3\Packages\

Copy the .sublime-syntax file
